About the Extension

Multi-Point Auto Clicker is a powerful Chrome extension designed for developers, testers, and power users. It allows you to automate repetitive clicking tasks by setting multiple coordinates on any webpage with individual delay timers.

Core Features:

  • Easy Multi-Point Selection (Point 1, 2, 3...)
  • Customizable Time Intervals for each point.
  • Dedicated Sidebar for seamless management.
  • Visual 'crosshair' cursor for pinpoint accuracy.
  • Keyboard shortcut (ESC) to stop or reset.
